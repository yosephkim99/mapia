# mapia

## 개요
tcp/ip 기반이면 상호간의 무언가를 전달하고 받는거라 생각을하다
한번쯤 들어봤을 마피아 게임이 생각나 마피아 게임을 만들면 프로그래밍 공부도 좀 되고
tcp/ip 수업에 배운 기법들을 한 번씩 다 사용해 볼 수 있을 것 같아 만들어봤다.
